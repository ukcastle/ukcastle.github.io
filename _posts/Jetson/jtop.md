jtop은 기존 리눅스의 `top` 명령어를 좀 더 시각적으로 보기 좋게 만들어진 jetson만의 파이썬 애플리케이션이다.

[공식 Github](https://github.com/rbonghi/jetson_stats)를 보면 설치 방법이 매우 쉽게 되어있다.

```
sudo -H pip install -U jetson-stats
```
